STORYTELLER

About:

Storyteller is a micro engine for visual novels that runs on a terminal (prompt/console/tty). It’s text-based and has its own syntax.

Features:

  • Supports image files (currently only ppm and pgm).
  • It’s easy to write your stories, no need to programming anything.

General Help:

  • The story must be saved on the same folder as the executable with the name “story.txt”.
